What is MyGround?

A self-hosting platform built as a single Rust binary. Install it on any Linux machine and replace your dependency on Google, Apple, and Microsoft — photos, videos, music, files, ad-blocking, all under your control.

CLI-first, AI-agent friendly, MIT-licensed. Not a Docker container managing Docker. Not a Node.js monolith. Just one binary, your data, your rules.

Why MyGround
01

Single Binary

One Rust binary with embedded web UI. No containers-managing-containers complexity.

02

Tailscale Built-in

Every service gets its own Tailscale node with automatic HTTPS via MagicDNS.

03

Automated Backups

Restic integration from day one. S3, B2, Wasabi, local — scheduled and reliable.

04

Disk Management

Pick storage paths per service. SMART monitoring. Never run out of space by surprise.

05

CLI-First

Fully scriptable. Manageable by AI agents. Web UI for humans, CLI for automation.

06

Truly Open

MIT licensed. No PolyForm restrictions, no vendor lock-in, no surprises.
